Centering a Div with Margin Auto

This page demonstrates how to center a div horizontally using margin: 0 auto.

The Technique

To center a div horizontally using margin auto:

  1. Set a width on the element (can be pixels, percentage, or max-width)
  2. Set margin-left: auto and margin-right: auto
  3. Or simply use the shorthand: margin: 0 auto
  4. The element must be a block-level element (default for div)

Demo 1: Fixed Width

The green box below is centered using a fixed width of 300px:

I'm Centered!

This box uses margin: 0 auto with a fixed width of 300px.

.centered-box {
  width: 300px;
  margin: 0 auto;
}

Demo 2: Percentage Width with Max-Width

The red box below is centered using 60% width with a max-width of 500px:

Responsive Centering

This box uses width: 60% and max-width: 500px. Try resizing the browser!

.centered-box-percentage {
  width: 60%;
  max-width: 500px;
  margin: 0 auto;
}

Demo 3: Vertical Spacing

You can add vertical margins while keeping horizontal centering:

With Vertical Spacing

This box uses margin: 50px auto to add vertical spacing.

.centered-box-vertical {
  width: 300px;
  margin: 50px auto; /* 50px top/bottom, auto left/right */
}

How It Works

  • margin-left: auto and margin-right: auto distribute the remaining horizontal space equally on both sides
  • This only works horizontally - margin auto does NOT center vertically
  • The element must have a defined width (not 100%)
  • The element must be a block-level element
  • For inline or inline-block elements, use display: block first

When to Use Margin Auto

Best for:

  • Centering content containers horizontally
  • Page layouts and content wrappers
  • Simple, predictable horizontal centering
  • When you don't need vertical centering

Not suitable for:

  • Vertical centering (use flexbox, position absolute, or grid instead)
  • Centering inline elements (use text-align: center on parent)
  • Complex layouts with multiple centering requirements

Common Variations

/* Just horizontal centering */
margin: 0 auto;

/* With vertical spacing */
margin: 20px auto;

/* Different top/bottom margins */
margin: 20px auto 40px auto;

/* Explicit left/right */
margin-left: auto;
margin-right: auto;

Centering a Div with Position Absolute

This page demonstrates how to center a div using position: absolute without flexbox.

The Technique

To center a div using position absolute, follow these steps:

  1. Set the parent container to position: relative
  2. Set the child element to position: absolute
  3. Position it at top: 50% and left: 50%
  4. Use transform: translate(-50%, -50%) to offset it back by half its own width and height

The CSS Code

/* Parent container */
.demo-container {
  position: relative;
  width: 100%;
  height: 500px;
}

/* Centered child element */
.centered-box {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  width: 300px;
  padding: 30px;
}

Why This Works

  • position: relative on the parent creates a positioning context
  • position: absolute on the child removes it from normal flow
  • top: 50%; left: 50% positions the top-left corner of the element at the center
  • transform: translate(-50%, -50%) shifts the element back by half its own dimensions, truly centering it

Alternative Method (Fixed Dimensions)

If you know the exact dimensions of the element, you can also use negative margins:

.centered-box {
  position: absolute;
  top: 50%;
  left: 50%;
  width: 300px;
  height: 200px;
  margin-left: -150px; /* Half of width */
  margin-top: -100px; /* Half of height */
}

Note: The transform method is preferred because it works regardless of the element's dimensions.

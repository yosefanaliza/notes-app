Help & Documentation

Learn how to use NotesApp effectively!

Getting Started

Click the "+ New Note" button in the sidebar to create a new note. Give it a title and start typing in the text area.

Organizing Notes

Use categories and tags to organize your notes. Select a category from the dropdown menu and add custom tags to make finding notes easier.

Saving & Deleting

Use the "Save" button to save your changes. The "Delete" button removes the current note permanently.

Common Questions

Q: How do I edit an existing note?

A: Click on any note in the sidebar to view and edit it in the main editor area.

Q: Can I change the category of a note?

A: Yes! Use the category dropdown in the editor toolbar to change it.

Q: What are tags for?

A: Tags help you organize and search for notes by topics or keywords.
